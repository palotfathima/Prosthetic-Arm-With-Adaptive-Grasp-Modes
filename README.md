# Prosthetic-Arm-With-Adaptive-Grasp-Modes
The PROSTHETIC ARM WITH ADAPTIVE GRASP MODES is designed to empower individuals with limb loss by enhancing their ability to manage
everyday tasks independently, specifically within a dining environment. The system
combines vision-based object recognition and dynamic grip control to facilitate safe
and efficient interaction with common dining items.

A camera captures real-time images that are processed to identify tableware
such as cups, bowls and spoons, and provisions like apples. Based on this visual data,
the system adapts the prosthetic arm’s grasp, depending on the type of object. Force
sensors continuously provide feedback on grip strength, allowing the prosthetic arm
to adjust its pressure dynamically, ensuring secure handling without risk of damage.

This project presents a functional and accessible prosthetic solution that combines advanced image processing and sensor feedback to restore autonomy in daily
dining tasks for individuals with upper limb loss.
